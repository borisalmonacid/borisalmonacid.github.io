Andean Condor Algorithm ======= Andean Condor Algorithm
The Algorithm of the Andean Condor is a metaheuristic inspired by nature, specifically in the pattern of movement of the Andean condor when searching for its food. The Algorithm of the Andean Condor presents a balance of its population through a performance indicator. This balance determines the quantity of Andean condors that will carry out an exploration or intensification movement. The Algorithm of the Andean Condor has a flexible design, which allows easy integration of heuristics or specific procedures according to the optimisation problem to be solved. >>>>>>> origin

<<<<<<< HEAD AutoMH - Automatically Create Evolutionary Metaheuristic Algorithms Using Reinforcement Learning ======= Machine learning research has been able to solve problems in multiple aspects. An open area of research is machine learning for solving optimisation problems. An optimisation problem can be solved using a metaheuristic algorithm, which is able to find a solution in a reasonable amount of time. However, there is a problem, the time required to find an appropriate metaheuristic algorithm, that would have the convenient configurations to solve a set of optimisation problems properly. A solution approach is shown here, using a proposal that automatically creates metaheuristic algorithms aided by a reinforced learning approach. Based on the experiments performed, the approach succeeded in creating a metaheuristic algorithm that managed to solve a large number of different continuous domain optimisation problems. This work’s implications are immediate because they describe a basis for the generation of metaheuristic algorithms in real-time. >>>>>>> origin
